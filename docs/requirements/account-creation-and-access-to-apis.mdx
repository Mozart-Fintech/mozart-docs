Para acceder a nuestras APIs, es necesario contar con una cuenta autorizada y un token de acceso, los cuales son proporcionados directamente por Mozart. A continuación, se detalla el proceso para solicitar acceso y comenzar a utilizar nuestras APIs.

<Admonition type="important" title="Solicitud de Cuentas">
Las cuentas de usuario no se pueden crear de manera independiente y deben ser solicitadas a nuestro equipo. Para solicitar una cuenta, por favor contacte a cvaldomir@mozartfintech.com con los detalles de su empresa y el tipo de integración que desea realizar.
</Admonition>

<Admonition type="important" title="Tokens de Acceso">
Junto con la creación de su cuenta, se le proporcionará un token de acceso único. Este token es esencial para realizar llamadas autenticadas a nuestras APIs y debe ser manejado con la máxima seguridad.
</Admonition>

Una vez que haya recibido su cuenta y token de acceso, podrá consultar la sección de **Referencia de la API** para obtener más detalles sobre cómo utilizarlos en sus solicitudes.
